    How to Use

    1. Accessing the Application: Open the ERP Dashboard through the provided URL using a modern browser (e.g., Chrome, Edge, Firefox).

    2. Dashboard Overview: You will be directed to the main Dashboard which serves as the entry point to all modules.

    3. Sidebar Navigation: Use the sidebar on the left to navigate between Dashboard, Components, Utilities, Pages, Charts, Tables, and Help sections.

    4. Interacting with Modules: Click each menu to display its contents — for example, tables to view data, charts for visual insights, or components for reusable UI.

    5. Responsive Interface: This system is optimized for mobile, tablet, and desktop, ensuring ease of use across devices.

    6. Need Help? Return to this “How to Use” tab for guidance on interacting with the system.

    Key Features

    • Modular Sidebar Navigation: Allows users to quickly access components, pages, utilities, and data tables.
    • Responsive Layout: Works seamlessly across desktops, tablets, and mobile devices.
    • Design System: Unified and reusable UI components styled with Bootstrap and customized with Laravel Blade templating.
    • Tab Navigation: Clean separation of interface sections using tabbed panels.
    • Chart & Table Integration: Display dynamic data visualizations and records.

    Dashboard Overview

    This ERP Dashboard application provides a unified interface for accessing various modules such as components, utilities, pages, charts, and tables. It is designed to help internal users manage information efficiently across departments using a consistent design system.
